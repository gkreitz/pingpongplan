# PingPongPlan

A simple, interactive tool to help schedule table tennis tournaments (specifically tailored for how they are played in Sweden).

Try it out live: [https://gkreitz.github.io/pingpongplan/](https://gkreitz.github.io/pingpongplan/)

## Features

- **Schedule Grid**: Visualize the tournament day by available time slots.
- **Classes**: Configure different tournament classes (e.g., Boys U11).
- **Match Blocks**: Define match blocks based on classes, required number of tables, and duration.
- **Drag & Drop**: Easily place and adjust unscheduled blocks directly on the schedule timeline.

## Local Development

The project operates entirely client-side using Vanilla HTML, CSS, and JavaScript.

To run it locally:
1. Clone the repository.
2. Open `index.html` in your web browser. No local web server or build process is strictly required, although you can use one (e.g., `npx serve`) if preferred.
